\## Advanced Screener Guide



\### Derived Metrics

\- PEG = PE / EPS Growth

\- Debt-to-FCF = Total Debt / Free Cash Flow



\### Temporal Rules

\- "Positive EPS last 4 quarters" means EPS > 0 for each of last 4 reported quarters



\### Export

\- CSV export includes matched filters, derived metrics, and timestamp



\### Limitations

\- Missing quarterly data is excluded from derived metrics
