# Validation Rules

## Rule Validity
- Detect unsatisfiable conditions
- Conflicting constraints rejected

## Ambiguity Detection
- Missing time window
- Undefined aggregation (ANY vs ALL)

## Safety Checks
- Divide by zero prevention
- Negative denominator rejection

## Data Availability
- Required periods must exist
- Sparse data flagged

## Error Types
- VALIDATION_ERROR
- DATA_ERROR
- SYSTEM_ERROR
