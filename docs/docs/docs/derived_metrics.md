# Derived Metrics

## Valuation
PEG Ratio = PE / EPS_Growth

## Leverage
Debt_to_FCF = Total_Debt / Free_Cash_Flow

## Growth
EPS_CAGR = (EPS_end / EPS_start)^(1/n) - 1
Revenue_CAGR = (Rev_end / Rev_start)^(1/n) - 1

## Efficiency
FCF_Margin = Free_Cash_Flow / Revenue

## Rules
- Denominator must be > 0
- Version controlled formulas
- Logged before execution
