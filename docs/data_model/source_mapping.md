# Source Mapping

## Market API (Exchange)
Fields:
- symbol, open_price, high_price, low_price, close_price, adj_close_price, volume, listing_date, sector, industry

## Fundamentals API
Fields:
- total_revenue, ebitda, net_income, free_cash_flow, total_assets, total_liabilities, eps_basic, eps_diluted

## Analyst API
Fields:
- analyst_consensus_rating, target_price_low, target_price_avg, target_price_high, num_analysts, estimate_confidence

## Corporate Actions API
Fields:
- dividend_announced, dividend_amount, dividend_ex_date, split_announced, split_ratio, buyback_announced, buyback_size

## Earnings Calendar API
Fields:
- next_earnings_date, last_earnings_date, last_reported_eps

## News API (optional)
Fields:
- latest_news_headline, news_sentiment_score, news_count_30d
