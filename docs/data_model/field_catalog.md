This document defines the initial set of data fields supported by the
AI-Powered Stock Screener.

## 1. Stock Metadata
| Field | Description |
|------|------------|
| symbol | Stock ticker symbol |
| company_name | Full company name |
| sector | Sector (IT, Banking, Pharma, etc.) |
| industry | Sub-industry |
| exchange | NSE / BSE |
| market_cap | Market capitalization |

## 2. Valuation Metrics
| Field | Description |
|------|------------|
| pe_ratio | Price-to-Earnings ratio |
| peg_ratio | Price/Earnings to Growth ratio |
| pb_ratio | Price-to-Book ratio |

## 3. Growth Metrics
| Field | Description |
|------|------------|
| revenue_growth_yoy | Year-over-year revenue growth |
| earnings_growth_yoy | Year-over-year earnings growth |

## 4. Profitability Metrics
| Field | Description |
|------|------------|
| revenue | Total revenue |
| ebitda | Earnings before interest, tax, depreciation |
| net_profit | Net profit |
| roe | Return on equity |

## 5. Financial Strength
| Field | Description |
|------|------------|
| free_cash_flow | Free cash flow |
| total_debt | Total debt |
| debt_to_fcf | Debt to free cash flow ratio |

## 6. Analyst & Events
| Field | Description |
|------|------------|
| analyst_rating | Buy / Hold / Sell |
| price_target_low | Analyst low price target |
| price_target_avg | Analyst average price target |
| price_target_high | Analyst high price target |
| buyback_announced | Whether buyback announced |
| next_earnings_date | Next earnings call date |
