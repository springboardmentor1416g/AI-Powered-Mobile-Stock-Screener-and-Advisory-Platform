# Advanced Screener User Guide

## Derived Metrics
- PEG = PE / EPS Growth
- Debt-to-FCF = Total Debt / Free Cash Flow

## Temporal Rules
"Positive EPS last 4 quarters" means EPS > 0 in each of the last 4 reported quarters.

## Reading Trends
↑ Increasing  
↓ Decreasing  
→ Flat

## Export
Results can be exported as CSV with metrics and timestamps.

## Limitations
- Stocks with missing EPS are excluded
- Derived metrics are computed using latest available data
