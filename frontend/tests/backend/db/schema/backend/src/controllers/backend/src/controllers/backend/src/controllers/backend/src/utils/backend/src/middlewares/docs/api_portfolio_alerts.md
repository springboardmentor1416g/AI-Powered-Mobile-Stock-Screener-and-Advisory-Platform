# Portfolio, Watchlist & Alert APIs

## Portfolio
POST /api/portfolio/add
PUT /api/portfolio/update
DELETE /api/portfolio/remove
GET /api/portfolio

## Watchlist
POST /api/watchlist/add
DELETE /api/watchlist/remove
GET /api/watchlist

## Alerts
POST /api/alerts
PUT /api/alerts/:id
PATCH /api/alerts/:id/status
DELETE /api/alerts/:id
GET /api/alerts
