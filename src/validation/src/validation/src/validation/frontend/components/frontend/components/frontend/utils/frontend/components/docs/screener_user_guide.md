# Advanced Screener User Guide

## Overview
This screener supports derived metrics and temporal rules.

## Supported Derived Metrics
- PEG Ratio
- Debt-to-Free Cash Flow
- PE Ratio

## Temporal Rules
- Positive EPS for last N quarters
- Revenue growth over TTM

## How to Use
1. Select advanced filters
2. Run screener
3. Tap company for fundamentals
4. Export or save results

## Limitations
- Data depends on backend availability
- Missing values handled gracefully
